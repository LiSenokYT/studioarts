# Как обновить проект на GitHub

## Простые команды для обновления:

```bash
# 1. Добавить все изменения
git add .

# 2. Создать коммит с описанием изменений
git commit -m "Исправлены все ошибки production: уведомления, изображения, формы"

# 3. Отправить изменения на GitHub
git push
```

Или если нужно указать ветку явно:
```bash
git push origin main
```

---

## Если возникнут проблемы:

### Проблема: "Updates were rejected"
Это значит, что на GitHub есть изменения, которых нет у тебя локально.

**Решение:**
```bash
# Скачать изменения с GitHub и объединить
git pull origin main

# Потом отправить свои изменения
git push origin main
```

---

### Проблема: "Please tell me who you are"
Нужно настроить Git.

**Решение:**
```bash
git config --global user.email "твой@email.com"
git config --global user.name "Твоё Имя"
```

---

### Проблема: Конфликты при git pull
Если есть конфликты, Git покажет какие файлы.

**Решение:**
```bash
# Вариант 1: Использовать свои изменения (перезаписать то что на GitHub)
git push origin main --force

# Вариант 2: Отменить локальные изменения и взять с GitHub
git reset --hard origin/main
git pull origin main
```

⚠️ **Внимание:** `--force` перезапишет изменения на GitHub твоими локальными файлами!

---

## Проверка статуса

Перед отправкой можно проверить что изменилось:

```bash
# Посмотреть какие файлы изменены
git status

# Посмотреть что именно изменилось в файлах
git diff

# Посмотреть историю коммитов
git log --oneline
```

---

## Быстрая шпаргалка

```bash
# Обновить проект на GitHub (3 команды):
git add .
git commit -m "Описание изменений"
git push

# Скачать изменения с GitHub:
git pull

# Посмотреть статус:
git status

# Отменить все локальные изменения:
git reset --hard

# Принудительно отправить (осторожно!):
git push --force
```

---

## Что происходит после git push?

1. Изменения отправляются на GitHub
2. Vercel автоматически видит новый коммит
3. Vercel начинает новый деплой
4. Через 1-2 минуты сайт обновится

Можешь следить за процессом на https://vercel.com/dashboard
